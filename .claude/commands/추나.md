# 추나 일별 입력

매일 진료 종료 후 원장별 추나 횟수를 입력받아 CSV에 기록합니다.

## 입력 형식 (자유롭게)

- "노왕식 건보8 ta5, 이문환 건보6 ta4, 방민준 건보7 ta3"
- "노왕식 8/5 이문환 6/4 방민준 7/3"  ← 건보/TA 순서
- "어제 노왕식 건보8 ta5 ..."  ← 날짜 명시 가능
- 일부 원장만 입력도 허용

날짜를 명시하지 않으면 오늘 날짜로 자동 기록.

## CSV 파일 위치 및 구조

파일: `c:\Users\하슬라한의원\한의원지표\data\추나현황.csv`

```
날짜,원장명,건보추나,TA추나
2026-05-10,노왕식,8,5
2026-05-10,이문환,6,4
2026-05-10,방민준,7,3
```

원장 이름이 바뀌어도 그냥 새 이름으로 입력하면 됨. 열 구조 변경 불필요.

## 처리 절차

1. 입력 파싱: 원장명 / 건보추나 수 / TA추나 수 추출
2. 날짜 결정: 오늘 날짜 또는 명시된 날짜
3. 기존 데이터 확인: 같은 날짜 + 같은 원장명 행이 있으면 업데이트, 없으면 추가
4. PowerShell로 CSV 업데이트 실행
5. 결과 표 출력

## PowerShell 실행 코드 (Claude가 실제 값을 채워서 실행)

```powershell
$csvPath = "c:\Users\하슬라한의원\한의원지표\data\추나현황.csv"
$entries = @(
    @{날짜="{{DATE}}"; 원장명="노왕식"; 건보추나="{{N_건보}}"; TA추나="{{N_TA}}"},
    @{날짜="{{DATE}}"; 원장명="이문환"; 건보추나="{{L_건보}}"; TA추나="{{L_TA}}"},
    @{날짜="{{DATE}}"; 원장명="방민준"; 건보추나="{{B_건보}}"; TA추나="{{B_TA}}"}
)

$data = @(Import-Csv $csvPath -Encoding UTF8)
foreach ($entry in $entries) {
    $idx = 0..($data.Count-1) | Where-Object { $data[$_].날짜 -eq $entry.날짜 -and $data[$_].원장명 -eq $entry.원장명 }
    if ($null -ne $idx) {
        $data[$idx].건보추나 = $entry.건보추나
        $data[$idx].TA추나 = $entry.TA추나
    } else {
        $data += [PSCustomObject]$entry
    }
}
$data | Export-Csv $csvPath -Encoding UTF8 -NoTypeInformation
```

## 출력 형식

```
✅ 2026-05-10 추나 기록 완료

| 원장   | 건보추나 | TA추나 | 합계 |
|--------|---------|--------|------|
| 노왕식 | 8       | 5      | 13   |
| 이문환 | 6       | 4      | 10   |
| 방민준 | 7       | 3      | 10   |
| 합계   | 21      | 12     | 33   |
```
